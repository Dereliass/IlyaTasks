Работа с XML


Задание:

Создать класс, в котором будет реализована обработка XML и его валидация.
Под валидацией подразумевается проверка XML файлов на закрытость тэгов, во избежание SAXException.
В случае невалидности подготавливает осмысленное сообщение в какой из строк допущена ошибка и в каком тэге, а так же проверяющей функцией возвращается false.
ЯП - Java. Задание необходимо будет оформить в виде Pool request к https://github.com/CelticBoozer/IlyaTasks.
Код должен пройти проверки CheckStyle plugin для IDEA, с учетом Google codestyle. А так же внутренний codereview.
Способ реализации свободный.

Пример:

<customized>d= a<customized> - Не валидно.

<customized>d= a</customized> - Валидно.
